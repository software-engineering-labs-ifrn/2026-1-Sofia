# Calculadora no Terminal

Este projeto implementa uma calculadora interativa que funciona completamente pelo terminal, desenvolvida em Python com programação orientada a objetos (POO).

## O que a aplicação faz

A calculadora oferece um menu simples para realizar operações básicas de matemática, incluindo:

- Soma
- Subtração
- Multiplicação
- Divisão
- Raiz quadrada
- Potência (x²)

Também trata erros comuns, como:

- Divisão por zero
- Entradas inválidas
- Raiz quadrada de números negativos

## Estrutura do projeto

- main.py: contém a implementação completa da calculadora no terminal, incluindo a lógica de cálculo e a interação com o usuário.

## Como executar

1. Certifique-se de que o Python esteja instalado em seu sistema.
2. Abra o terminal na pasta do projeto.
3. Execute o comando:

```bash
python main.py
```

## Tecnologias utilizadas

- Python 3
- Math

## Observações

Este projeto foi desenvolvido como uma demonstração de uma aplicação simples, leve e fácil de executar em qualquer ambiente que suporte terminal.
